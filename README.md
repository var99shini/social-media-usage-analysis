# Social Media Usage Analysis

This project analyzes social media usage patterns using Python and Pandas. 
Visualizations are created to study app-wise, day-wise, and purpose-based usage trends.

## Insights
- Instagram accounts for the highest total usage time.
- Social media usage increases during weekends.
- Entertainment is the dominant purpose of usage.
- YouTube shows significant usage for learning purposes.
